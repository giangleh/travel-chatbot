# User Personas

## Persona 1: The Traveler (Primary)
- **Name**: Alex
- **Role**: End user exploring Tokyo using the curated list
- **Goals**: Discover spots nearby, get day plans, navigate efficiently
- **Behaviors**: Uses chat on mobile while walking; asks quick questions; wants instant actionable answers with maps
- **Frustrations**: Too many options without context; broken links; slow responses; having to switch between apps
- **Context**: On the ground in Tokyo with mobile data, limited time per neighborhood

## Persona 2: The Planner (Secondary)
- **Name**: Sam
- **Role**: User planning a trip before departure
- **Goals**: Build multi-day itineraries, understand neighborhood groupings, estimate time needed
- **Behaviors**: Uses web on desktop; asks broader questions; wants to see full day layouts
- **Frustrations**: Disorganized recommendations; no sense of proximity; can't visualize routes
- **Context**: At home before the trip, has time to explore and organize

## Persona 3: The Curator (Admin)
- **Name**: Giang (you)
- **Role**: List owner who maintains the Master List
- **Goals**: Add/remove/update spots, keep data accurate, manage categories
- **Behaviors**: Adds spots via chat ("add this to my list"); updates hours/ratings when things change
- **Frustrations**: Having to open Airtable separately; losing track of what's been added
- **Context**: Both pre-trip (research) and during-trip (discovering new spots)
